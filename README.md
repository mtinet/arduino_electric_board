# electric_board

* 조종기  

|아두이노 | 포텐시오미터|
| :------: | :-------: |
|GND     |   GND  |
|5V      |   VCC  |
|A0      |   Sig  |

* 수신기  

| 아두이노 | 서보모터 |
| :------: | :-------: |
|GND     |   GND  |
|5V      |   VCC  |
|D9      |   Sig  |

* 공통  

| 아두이노 | 블루투스 |
| :------: | :-------: |
|GND     |   GND  |
|5V      |   VCC  |
|A0      |   Sig  |
